# Modules Overview

## Module 1: Introduction to the Web

**Units:**

- Unit 1.1 – What is the Web & how does it work? (pages, browsers, servers)
- Unit 1.2 – Basic structure of a web page (HTML document structure, head/body, tags)
  **Key activities:**
- Create a simple webpage with title, heading, paragraph
- Explore view-source, inspect element, browser dev tools
  **Outcomes:** Students and teachers understand the basics of the web environment; know how HTML fits into that.

---

## Module 2: HTML Foundations

**Units:**

- Unit 2.1 – Building content: headings, paragraphs, lists, links, images
- Unit 2.2 – Structuring for access: semantic tags (section, article, nav), forms, tables
  **Key activities:**
- Create a multi-section webpage (e.g., small portfolio or topic page)
- Add a form for user input (name, email) and table for data presentation
  **Outcomes:** Learners can construct well-formed HTML documents, apply semantics and basic accessible structure.

---

## Module 3: CSS Styling & Layout

**Units:**

- Unit 3.1 – Applying styles: selectors, colours, typography, box-model
- Unit 3.2 – Layout & responsive design: Flexbox, Grid, media queries
  **Key activities:**
- Style the HTML from Module 2 with custom colours, fonts, spacing
- Make the page responsive so it adapts to mobile and desktop layouts
  **Outcomes:** Learners can style web pages and adapt layout for different devices.

---

## Module 4: JavaScript Essentials

**Units:**

- Unit 4.1 – Programming basics: variables, conditions, loops, functions
- Unit 4.2 – DOM manipulation & events: handle user input, change page content dynamically
  **Key activities:**
- Write a JavaScript function to validate the form created in Module 2
- Create an interactive element (e.g., button that changes content, list filter)
  **Outcomes:** Learners gain foundational programming skills and understand how JS connects with webpage content.

---

## Module 5: Data & Visualization

**Units:**

- Unit 5.1 – Working with data: arrays, JSON, external data sources
- Unit 5.2 – Visual representation: simple charts, canvas, interactive widgets
  **Key activities:**
- Load sample data (e.g., environmental data) and display in table/list
- Use a chart library (or vanilla canvas) to visualise the data on the webpage
  **Outcomes:** Learners can incorporate data into their web artefacts and visualize it for insight.

---

## Module 6: Creative Web Projects

**Units:**

- Unit 6.1 – Design thinking with code: ideation, prototyping, building a web artefact
- Unit 6.2 – Peer review & iteration: sharing, receiving feedback, refining project
  **Key activities:**
- In groups or pairs, design a mini web project linking a STEAM theme (e.g., sustainability, physics experiment)
- Present the project, review peer work, iterate improvements
  **Outcomes:** Learners apply combined skills from previous modules in a creative, collaborative context.

---

## Module 7: Green STEAM Challenge

**Units:**

- Unit 7.1 – Web App for sustainability: apply skills to a chosen SDG or STEAM-theme
- Unit 7.2 – Presentation & reflection: share project, reflect on learning and digital competence growth
  **Key activities:**
- Develop a simple web app or interactive webpage that addresses a sustainability question
- Present the app to classmates or teachers, write a self-reflection mapped to competence map
  **Outcomes:** Learners demonstrate their digital proficiency in a meaningful STEAM context; reflection connects to the competence map.
