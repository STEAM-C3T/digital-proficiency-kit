# Module 1: Introduction to the Web

- Examples
  - [minimal-page.html](./examples/minimal-page.html) — A minimal semantic HTML5 page to inspect and extend.
- Tasks
  - [Task: Your First Webpage](./tasks/task-1-first-webpage.md) — Build a simple, accessible page with semantic tags.
