# Teacher Notes – Module 1: Introduction to the Web

Adaptation options:

- Simplify: Provide a ready template and ask learners to change title/heading.
- Extend: Introduce developer tools and network tab briefly.
- Accessibility: Discuss semantics and why screen readers benefit from structure.

Assessment suggestions (DigComp – Information, Communication):

- Criteria: Correct structure, clarity of content, understanding of web request flow.
- Evidence: Minimal page, diagram/sketch of request-response.
- Feedback: Reinforce value of semantic tags; suggest improving headings/structure.
