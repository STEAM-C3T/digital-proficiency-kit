# Task: Your First Webpage

Design goals:

- Create a semantic page with `<header>`, `<main>`, and `<footer>`.
- Include a title, a main heading, and one paragraph.

Acceptance criteria:

- Valid HTML5 structure with `<!DOCTYPE html>` and `lang` attribute.
- Uses semantic tags (no presentation-only `<div>` for structure).
- Page readable on mobile (no horizontal scroll at 320px).

Deliverable:

- One HTML file. Include a comment at the top explaining what the page is about.
