# Teacher Notes – Module 5: Data & Visualization

Adaptation options:

- Simplify: Provide an array and a partially complete drawing routine.
- Extend: Add axes, gridlines, or tooltips on hover; compare two data series.
- Accessibility: Pair the chart with an HTML table containing the same data.

Assessment suggestions (DigComp 2.2 – Information, Content Creation):

- Criteria: Correct data mapping, clarity of labels, visual readability.
- Evidence: Working chart and a short explanation of scaling.
- Feedback: Ask how the visual could be made clearer; suggest adding legends or annotations.
