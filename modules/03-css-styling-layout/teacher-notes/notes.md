# Teacher Notes – Module 3: CSS Styling & Layout

Adaptation options:

- Simplify: Provide a base HTML template and a palette to choose colours from.
- Extend: Introduce CSS variables and media queries for responsive adjustments.
- Accessibility: Discuss colour contrast and focus-visible states; encourage testing with keyboard only.

Assessment suggestions (map to DigComp 2.2 – Content Creation, Safety):

- Criteria: Visual hierarchy, readability, responsive behaviour, accessibility.
- Evidence: Styled HTML page, brief reflection on design choices.
- Feedback: Highlight effective use of classes and spacing; suggest improvements for contrast or structure.
