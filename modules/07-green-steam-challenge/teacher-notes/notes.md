# Teacher Notes – Module 7: Green STEAM Challenge

Adaptation options:

- Simplify: Turn app into a single-screen checklist with clear scoring.
- Extend: Collect (non‑personal) aggregate data for class discussion.
- Ethics: Discuss privacy and data minimisation; keep all data local.

Assessment suggestions (DigComp 2.2 – Problem Solving, Safety, Content Creation):

- Criteria: Relevance to theme, functionality, usability, reflection.
- Evidence: Working prototype, explanation of impact, user feedback notes.
- Feedback: Encourage user testing and iteration; consider accessibility and inclusivity.
