# Teacher Notes – Module 6: Creative Web Projects

Adaptation options:

- Simplify: Limit to colour and typography changes; remove animation.
- Extend: Add audio-reactive elements or data-driven variation.
- Accessibility: Ask students to provide alt text/captions describing the visual behaviour.

Assessment suggestions (DigComp 2.2 – Content Creation, Communication):

- Criteria: Concept clarity, design execution, interaction design, reflection quality.
- Evidence: Working poster, caption/statement, peer feedback notes.
- Feedback: Encourage iterative refinement; discuss balancing aesthetics and usability.
