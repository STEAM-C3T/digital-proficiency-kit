# Teacher Notes – Module 2: HTML Foundations

Adaptation options:

- Simplify: Provide a skeleton page with TODOs; learners fill content sections.
- Extend: Add internal navigation with anchors and a simple table of contents.
- Accessibility: Stress heading order, alt text, and label-input associations.

Assessment suggestions (DigComp – Content Creation, Safety):

- Criteria: Semantic structure, correctness of markup, accessibility basics.
- Evidence: Topic page and/or survey page; reflective comments on choices.
- Feedback: Suggest improvements to heading hierarchy and link text clarity.
