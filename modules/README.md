# Modules

This folder contains the learning modules of the Digital Proficiency Kit. Each module aims to be browser‑only (no installs), accessible, and classroom‑ready with:

- Units (lesson content and templates)
- Examples (runnable HTML/CSS/JS)
- Tasks (guided and independent practice with acceptance criteria)
- Teacher notes (adaptations, assessment suggestions)

## Module Index

1. Introduction to the Web — `modules/01-introduction-to-the-web/`
2. HTML Foundations — `modules/02-html-foundations/`
3. CSS Styling & Layout — `modules/03-css-styling-layout/`
4. JavaScript Essentials — `modules/04-javascript-essentials/`
5. Data & Visualization — `modules/05-data-visualization/`
6. Creative Web Projects — `modules/06-creative-web-projects/`
7. Green STEAM Challenge — `modules/07-green-steam-challenge/`

See also `docs/modules.md` for a narrative overview.
