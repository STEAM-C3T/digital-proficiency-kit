# Modules

This folder contains the learning modules of the Digital Proficiency Kit. Each module is browser‑only (no installs), accessible, and classroom‑ready with:

- Units (lesson content and templates)
- Examples (runnable HTML/CSS/JS)
- Tasks (guided and independent practice with acceptance criteria)
- Teacher notes (adaptations, assessment suggestions)

For a narrative overview, see `docs/modules.md`.

## Module Index & Summaries

1. [Introduction to the Web](./01-introduction-to-the-web/README.md)
   - Understand how the web works (client–server, URLs) and create a minimal, semantic HTML page with accessibility basics.
   - [Teacher Toolkit](https://github.com/STEAM-C3T/teacher-toolkit/tree/main/modules/01-introduction-to-the-web)
2. [HTML Foundations](./02-html-foundations/README.md)
   - Build well‑structured content with semantic tags, accessible forms (labels), and simple tables with headers and captions.
   - [Teacher Toolkit](https://github.com/STEAM-C3T/teacher-toolkit/tree/main/modules/02-html-foundations)
3. [CSS Styling & Layout](./03-css-styling-layout/README.md)
   - Apply typography, colour, and spacing; construct responsive layouts using Flexbox/Grid with attention to contrast and focus.
   - [Teacher Toolkit](https://github.com/STEAM-C3T/teacher-toolkit/tree/main/modules/03-css-styling-layout)
4. [JavaScript Essentials](./04-javascript-essentials/README.md)
   - Add interactivity with events and DOM updates; manage simple UI state and practice progressive enhancement.
   - [Teacher Toolkit](https://github.com/STEAM-C3T/teacher-toolkit/tree/main/modules/04-javascript-essentials)
5. [Data & Visualization](./05-data-visualization/README.md)
   - Map small datasets to visuals (Canvas/SVG), implement scaling, and pair charts with tables for accessibility.
   - [Teacher Toolkit](https://github.com/STEAM-C3T/teacher-toolkit/tree/main/modules/05-data-visualization)
6. [Creative Web Projects](./06-creative-web-projects/README.md)
   - Explore generative visuals and interactive posters; expose parameters via UI and document iterations.
   - [Teacher Toolkit](https://github.com/STEAM-C3T/teacher-toolkit/tree/main/modules/06-creative-web-projects)
7. [Green STEAM Challenge](./07-green-steam-challenge/README.md)
   - Build a small sustainability‑focused mini‑app aligned to SDGs; consider ethics, privacy, and impact communication.
   - [Teacher Toolkit](https://github.com/STEAM-C3T/teacher-toolkit/tree/main/modules/07-green-steAM-challenge)
